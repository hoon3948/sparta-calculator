계산기 프로그램  
1. 구현된 것  
  - 정수 입력(음의 정수 입력 가능)  
  - 부호 입력(+,-,*,/,%)  
  - 반복 작업(입력 받고 계산 반복)  
  - 계산 기록 호출 가능(계산 종료 후 list 입력)  
  - 계산 기록 삭제 가능(기록 호출 후 delete 입력, 원할때까지 반복)  
  - 나눗셈이나 나머지를 구할때 num2가 0인지 점검(0인 경우 다시)  
  - 정해진 부호 외에 입력시 재입력  
2. 추가 구현하고 싶은 목표  
  - 도전과제  
  - 정해진 부호 외에 입력시 기록에 남기지 않게 조정  
  - 정수 입력시 부호 같이 입력하면 재입력받을수있게 조정  
  - 코딩 간략화
